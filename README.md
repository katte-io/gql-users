# Express GraphQL App

A sample express App to demonstartae working with GraphQL, fetching data from a DB / API and using GraphiQL as a dev tool.

## Concepts covered

- GraphQL Schemas

- Root Queries

- Resolving Queries

- GraphiQL tool

- Handling cyclic dependencies

- Nested Queries

- Query Fragments

- GraphQL Types

- Non Null fields

- Mutations
